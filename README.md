



# Portfólio Pessoal

Este projeto é um portfólio desenvolvido em HTML e CSS, criado para apresentar minha identidade como desenvolvedor(a), minhas principais habilidades e projetos em destaque.

## Funcionalidades

- Layout moderno e responsivo.
- Seção de apresentação personalizada para mostrar quem sou.
- Tags de tecnologias que identificam minha stack de desenvolvimento.
- Cards de projetos que exibem imagem, título e descrição para facilitar a visualização.

## Tecnologias Utilizadas

- HTML5
- CSS3
